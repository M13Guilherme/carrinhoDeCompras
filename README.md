🛒 Carrinho de Compras
Aplicação web para simular um carrinho de compras, desenvolvida como exercício prático do curso de Lógica de Programação da Alura.
📋 Sobre o projeto
O Carrinho de Compras permite selecionar produtos, definir a quantidade desejada e adicioná-los ao carrinho, exibindo o subtotal de cada item e o valor total da compra. Também conta com um botão para limpar o carrinho e reiniciar a sessão.

O HTML e o CSS foram fornecidos pela Alura como material de apoio. O foco do exercício foi o desenvolvimento da lógica em JavaScript.

🚀 Funcionalidades

Selecionar produto e quantidade
Adicionar produto ao carrinho com subtotal calculado automaticamente
Exibir o valor total acumulado da compra
Limpar o carrinho e reiniciar o valor total
